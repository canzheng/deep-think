## Stage 8 - Signal Translation

Purpose:
- convert questions into observables

Should not:
- confuse signals with evidence hierarchy
- skip decision implications
- create monitoring noise with no update rule

STEP 16 - TRANSLATE QUESTIONS INTO SIGNALS

For each high-value question, specify:
- Question
- Observable signal
- Preferred evidence source
- Backup evidence source
- Frequency
- Threshold / update rule
- Belief update implication
- Confidence in the signal under the current uncertainty mode
- Decision implication:
  - changes action?
  - changes timing?
  - changes sizing?
  - changes staging?
  - changes hedge / exit / kill criteria?

- Separate evidence from signals, and signals from decisions.
- Distinguish narrative from diagnostic signals.
