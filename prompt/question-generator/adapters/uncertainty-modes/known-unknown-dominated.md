UNCERTAINTY MODE — KNOWN-UNKNOWN DOMINATED

Research behavior:
- quantify ranges
- identify thresholds
- keep scenario set relatively tight

Risk:
- false precision

## Stage Relevance

Routing — Primary
- Classifies the problem as one where key variables are visible but future values remain uncertain.

Boundary — Light
- Usually keeps scope relatively tight because the main variables are already visible.

Structure — Primary
- Supports higher confidence in the structural map and pushes attention to sensitivities and thresholds.

Scenarios — Primary
- Prefers tighter branches and narrower probability bands.

Question Generation — Primary
- Favors range, threshold, and sensitivity questions.

Evidence Planning — Primary
- Encourages direct measurement of visible variables and disciplined range updates.

Decision Logic — Primary
- Supports action logic tied to explicit thresholds while warning against false precision.

Signal Translation — Primary
- Signals should narrow ranges and confirm threshold movement.

Monitoring — Primary
- Monitoring should be threshold-based and relatively regular.

Render — Modulating
- Should show confidence ranges without pretending certainty.
