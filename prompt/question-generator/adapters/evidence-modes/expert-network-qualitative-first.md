EVIDENCE MODE — EXPERT-NETWORK / QUALITATIVE-FIRST

Prioritize:
- interviews
- channel checks
- customer behavior
- operator views
- partner feedback
- revealed field behavior

Strengths:
- fast
- useful for hidden frictions

Weaknesses:
- anecdotal bias
- sample bias
- triangulation required

## Stage Relevance

Routing — Primary
- Selects a qualitative and field-intelligence proof style.

Boundary — Light
- Can widen scope toward hidden frictions and off-balance-sheet behavior.

Structure — Modulating
- Allows more hypotheses about field behavior, friction, and hidden operational reality.

Scenarios — Modulating
- Prefers branches supported by repeated operator behavior and field evidence.

Question Generation — Primary
- Favors questions about what practitioners are seeing that is absent from official reporting.

Evidence Planning — Primary
- Sets triangulation rules, freshness bias, and bias controls for anecdotal evidence.

Decision Logic — Modulating
- Can support action when triangulated field behavior is more current than formal reporting.

Signal Translation — Primary
- Favors channel checks, customer behavior, partner feedback, and operator observations.

Monitoring — Primary
- Monitoring should watch repeated qualitative pattern shifts, not isolated anecdotes.

Render — Light
- Usually requires explicit caveats about sample bias and triangulation.
