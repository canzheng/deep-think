TASK ADAPTER — PREDICT

Prioritize:
- decisive stakeholders
- incentives under constraints
- endurance
- branch logic
- probabilities
- what can still change the path

## Stage Relevance

Routing — Primary
- Sets the overall job to forecasting what happens next.

Boundary — Primary
- Pulls scope toward actors, constraints, and variables that can still change the future path.

Structure — Primary
- Prioritizes actor, incentive, and constraint structure over retrospective causal narration.

Scenarios — Primary
- Requires explicit future paths, branch points, and rough probability logic.

Question Generation — Primary
- Favors probability-shifting questions and branch-distinguishing questions.

Evidence Planning — Modulating
- Prefers evidence that raises or lowers scenario odds rather than merely explaining the past.

Decision Logic — Modulating
- Strong when prediction is being used to support action, otherwise secondary.

Signal Translation — Primary
- Signals are mainly for updating path probabilities and branch confidence.

Monitoring — Primary
- Monitoring should track branch triggers, confirmation patterns, and belief updates.

Render — Primary
- Pushes the deliverable toward paths, odds, and update conditions.
